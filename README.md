# NutriDoc - AI Health Ingredient Co-Pilot

**NutriDoc** is an AI-powered medical ingredient analyzer designed to help users make safer food choices based on their specific health profiles. Built for Hackathons, it features a clinical-grade UI, smart ingredient detection, and localized Hinglish advice.

## üöÄ Key Features
- **Smart Scanner**: Automatically identifies food ingredient labels using OCR and filters out non-food items.
- **Medical Profile Context**: Tailored advice for Diabetes, High/Low Blood Pressure, Thyroid, PCOD, Kidney issues, and Pregnancy.
- **Clinical Impact Analysis**: Explains the "Internal Impact" and "Potential Consequences" of consuming specific ingredients in real-world terms.
- **Hinglish Advice**: Natural, easy-to-understand feedback in a mix of Hindi and English (e.g., "Kaisay asar karta hai?", "Kya ho sakta hai?").
- **Voice Feedback**: Real-time audio summaries for accessibility.
